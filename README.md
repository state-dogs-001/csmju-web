# CSMJU Client

## Installation

### Install dependencies

```
npm install
```

or

```
yarn install
```

### Copy .env.example to .env

```
cp .env.example .env
```

or

```
copy .env.example .env
```

### Run app

```
npm run dev
```

or

```
yarn dev
```

### Compiles and minifies for production

```
npm run build
```

or

```
yarn build
```
